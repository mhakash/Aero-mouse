Introduction

In this project we tried to develop a wireless mouse which we can use by motion control. It works similarly as wireless remotes found in modern television. It has 2 buttons for left and right click and a power button.

Equipments

  • Hardware

    • Atmega32
    • Mpu-6050
    • HC-05 bluetooth module

  • Software

    • A driver written in Java
    • Java

Principals

The working principles are quite simple. First when we press power on button the whole system gets power. Then the gyroscope sends raw data to Atmega32 unit. The microprocessor processes the data and send it to pc via bluetooth using serial communication.


In our pc we need to install a driver also written by us. The driver is written in Java. The driver catches the data provided by the microprocessor from a communication port. After that the program uses the data to do necessary work for our mouse movements and clicks. Following is the diagram.
